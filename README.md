# Horror Castle

A dark, first-person horror castle explorer built with React, TypeScript, Three.js, and Tailwind CSS.

## Controls

**Desktop**
- `W` `A` `S` `D` — walk
- Mouse — look around
- Click — capture cursor

**Mobile**
- Left half of screen — drag to move
- Right half of screen — drag to look

## Run

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually `http://localhost:5173`).
